My Personal Profile Website
About
This is a simple personal profile website built using HTML, CSS, and Bootstrap. It includes sections for About, Skills, Education, and Contact to showcase my background and expertise.

Features
✅ Responsive navigation bar
✅ Hero section with profile image and bio
✅ Sections for personal details, skills, and education
✅ Styled using Bootstrap and custom CSS
✅ Mobile-friendly design

Technologies Used
HTML5 - Structure of the website
CSS3 - Styling and layout
Bootstrap 5 - Responsive design and components
Installation & Usage
Clone this repository:

git clone https://github.com/Bida24-308/profile-website.git
Open the index.html file in a browser.
Customize the content by editing the HTML and CSS files.
Customization
Change profile image: Replace images.jpg
Modify text and links in index.html
Adjust styles in style.css
